## Alternativas

Si no puedes conseguir estas imágenes:
1. Usa sitios gratuitos como Unsplash, Pexels o Pixabay
2. Busca por los términos: "beauty products", "skincare routine", "hair care", etc.
3. Las imágenes placeholder actuales seguirán funcionando hasta que agregues las reales

## Rutas de Implementación

Todas las imágenes ya están referenciadas en `blog.js` con las rutas correctas:
- `img/blog/rutina-noche.jpg`
- `img/blog/vitamina-c.jpg`
- Etc.

Solo necesitas colocar los archivos en la carpeta correcta.
